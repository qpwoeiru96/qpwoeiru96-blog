这个问题最重要的是分段的问题，从右到左的排版方式是水平扩展，文章一旦过长容易超出屏幕的显示范围影响阅读。所以需要分段将其变成垂直扩展。

而这个恰恰是 CSS3 的 multi-column model 可以支持的。

Column boxes in a multi-column element are arranged into rows. Like table cells, the column boxes in a row are ordered in the inline direction of the multicol element.

以上是 w3 的说明： 行盒在多行元素里面排列成列，就像表格单元格一样，行盒在一列里面的排列方式是跟在这个多行元素的行内元素的方向一致的。最重要的是标粗的那一段，如果没有这一段我们无法实现分段的效果。

DEMO

虽然实现出来了，但是有一个地方需要注意的是，就是 <div class="classical-typing"> 的高度不是整个渲染区域的高度，而是单列的高度。从 w3 的介绍来看应该是多行元素里面的每一个行盒的高度都与这个多行元素的内容盒高度的使用值。反之亦然啊，所以比较难以控制此处的高度。特别是设置 background 的时候。

好了，古文排版的研究到此为止，已经基本上满足我之前所需要的效果了。
